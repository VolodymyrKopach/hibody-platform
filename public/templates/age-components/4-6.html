📋 Технічні характеристики

Увага: 15-20 хвилин максимум
Мінімальний розмір кнопок: 60x60px
Контрастність: мінімум 4.5:1
Підтримка: клавіатура, дотик, голосові команди
Анімації: плавні (0.4s), без швидких переходів
Адаптивність: планшети та десктопи
Візуальні покращення: градієнти, тіні, заокруглення, анімації
Звукові можливості: Web Speech API тільки для навчальних елементів
Мови: українська (пріоритет), англійська як резерв
Інтерактивність: всі компоненти повністю робочі з покращеним фідбеком
Геймифікація: система балів, прогрес-бар, нагороди, лічильник взаємодій
Збереження: автоматичне збереження прогресу в localStorage
Підказки: кнопка "💡 Підказки" для батьків та вчителів
Аналітика: відстеження навчальних взаємодій
Баланс: озвучка тільки там, де справді потрібно для навчання
