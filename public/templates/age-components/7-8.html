🤖 AI Generation Rules for 7-8 Years

MANDATORY REQUIREMENTS:

  • ✅ Interactive elements must be 50px+ for this age group (more precise control)
  • ✅ Support complex multi-step problem solving
  • ✅ ALL interactive elements must have onclick handlers
  • ✅ Use more sophisticated color schemes (less bright, more "adult")
  • ✅ Include progress tracking and detailed feedback
  • ✅ Minimum touch target size: 50px × 50px
  • ✅ Include comprehensive audio feedback for learning
  • ✅ ALWAYS include audio toggle button (audio-toggle class)
  • ✅ Respect audioEnabled global variable in all sound functions
  • ✅ Support sequential problem-solving workflows
  • ✅ Include mistake analysis and learning guidance

SLIDE TEMPLATE STRUCTURE FOR 7-8 YEARS:

  • 🏗️ ALWAYS use .slide-container as main wrapper
  • 📋 .slide-header: title area + meta info (lesson progress, time)
  • 📚 .slide-content: main educational content with .content-section blocks
  • 🎯 .activity-area: interactive exercises and games
  • 📊 .slide-footer: navigation controls + progress tracking
  • 🎨 .section-title: headers for content sections with emojis
  • ✅ .learning-objectives: bulleted learning goals
  • 🎮 .exercise-container: structured interactive activities

TYPOGRAPHY COMPONENTS FOR 7-8 YEARS:

  • 📚 .main-heading - Primary lesson titles (32px, gradient text)
  • 🎯 .section-heading - Major section headers (24px, underlined)
  • 📖 .sub-heading - Subsection titles (20px, medium weight)
  • 📝 .main-description - Main explanatory text (18px, readable)
  • 📄 .section-text - Regular body text (16px, optimal spacing)

  • 🌟 .highlight-text - Important information (yellow background)

STANDARDIZED IMAGE SIZES FOR AGES 7-8:

  • 🌟 Hero images (400×300): Main educational content, complex concepts
  • 📖 Story images (320×240): Narrative and contextual illustrations
  • 🎯 Activity images (240×240): Interactive learning exercises
  • ✨ Learning images (180×180): Supporting educational elements

AUDIO CONTROL REQUIREMENTS:

  • 🔊 ALWAYS include: <div class="audio-toggle" onclick="toggleAudio()"></div>
  • 📊 ALWAYS include: <div class="audio-status" id="audioStatus">Audio: ON</div>
  • 🎵 Check audioEnabled before all speech and tone functions
  • ⌨️ M key toggles audio globally
  • 👆 Click/touch audio button to toggle
  • 📱 Visual feedback when muted (red) vs active (blue)
  • 🔊 Include both tone generation and speech synthesis
  • 🎯 Age-appropriate voice settings (rate: 0.9, pitch: 1.0)
  • 📚 Educational content narration for learning support
  • 💡 Audio hints and guidance system
